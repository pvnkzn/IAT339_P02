P01 - Style Guide Template

A template you can use to start building out your own style guides with.

Interactive Elements Text Elements Combined Elements Product Listing Template Checkout Template Citations Branding

Interactive Elements

In-text link

CSS selector:

.in-text-link

Sample code:

<p>Check out our new items <a class = "in-text-link" href = "#in_text_link">here.</a></p>

Rendered element:

Check out our new items here.

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Shop</a>

Rendered element:

Shop

Submit/Reset Buttons

CSS selector:

form button

Sample code:

          
            <form method="post" action="#">
              <button type = "submit" value = "Submit">Submit</button>
              <button type = "reset" value = "Reset">Reset</button>
            </form>

Rendered element:

Text Input Field with Label

CSS selector:

form

Sample code:

          
            <form method="post" action="#">
              <label for="form">City:</label>
              <input type="text" id="form" placeholder="Type your city here">
            </form>

Rendered element:

            </form>

Checkout Cart

CSS selector:

.checkout-cart

Sample code:

          

            <h3>Checkout Cart</h3>

            <table class="checkout-cart">

              <thead>
                <tr>
                  <th>Product Image</th>
                  <th>Product Name</th>
                  <th>Price</th>
                  <th>Quantity</th>
                  <th>Subtotal</th>
                </tr>
              </thead>

              <tbody>
                <tr>
                  <td><img src="img/silver-heart-necklace.jpg" alt="A silver heart necklace." width = "200"></td>
                  <td>Silver Heart Necklace</td>
                  <td>$17.00</td>
                  <td>2</td>
                  <td>$34.00</td>
                </tr>

                <tr>
                  <td><img src="img/pink-dotted-sunglasses.jpg" alt="A pair of pink sunglasses." width = "200"></td>
